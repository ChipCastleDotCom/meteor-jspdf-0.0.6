meteor-jspdf

A meteor package for jsPDF.

Original library was converted for use with Meteor by Chip Castle Dot Com, Inc..

Installation

meteor add chipcastledotcom:jspdf

Documentation

Since this is only a wrapper package, please see the documentation at jsPDF for details on how to API usage.

Credits

Big thanks for to James Hall for providing MrRio/jsPDF as an Open Source package from which this package was created.
